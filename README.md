## KoBERT 언어모델을 사용한 한국어 지도 리뷰 평점 예측 (Starring Reviews_KR)

- 최적의 성능을 갖도록 모델 설계
- 실제 사용 가능한 클라이언트-서버 구조 어플리케이션 개발
